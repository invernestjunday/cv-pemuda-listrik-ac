# Pack 2 Overlay — Global Shell

Pack ini menambahkan global shell Astro:

- BaseLayout
- PageLayout
- Header
- MobileMenu
- Footer
- FloatingWhatsApp
- MobileBottomCTA
- WhatsAppButton
- MetaTags dasar
- Route 6 halaman utama
- Route dinamis halaman lokal listrik dan AC
- CSS sementara untuk shell

## Setelah copy overlay

Jalankan:

```cmd
npm run validate:content
npm run build
npm run preview
```

Jika PASS:

```cmd
git add .
git commit -m "pack-2: add global layout shell and whatsapp cta"
git push
```
